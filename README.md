# Video ASCII

Real-time video to ASCII art converter that runs entirely in your browser. Works on both desktop and mobile devices.

## URL

https://toaster496.github.io/video-ascii

## Features

- **Real-time ASCII conversion** from your camera via `getUserMedia`
- **Mobile-first** responsive design with touch-friendly controls
- **Front / back camera** toggle
- Multiple **character sets** (Standard, Detailed, Blocks, Binary)
- **Color modes**: Monochrome, Full Color, Matrix green
- **Adjustable resolution** density
- **Upload and convert** local video files
- **Snapshot** downloads of your ASCII creations
- Add to Home Screen (PWA) support

## How to use

1. Open the deployed URL on your phone or desktop
2. Tap **Start Camera**
3. Allow camera access when prompted
4. Use the controls to fine-tune the output
5. Tap **Snap** to save a PNG

## Tech Stack

Pure HTML5, CSS3, and vanilla JavaScript. No build tools or dependencies required. Deployed via GitHub Pages.
